# FFC-AI

Self-hosted mobile-controlled AI operations stack.

Primary target: Mattermost on a VPS plus a local/WSL AI runner.
Fallback target: Matrix/Synapse.

Telegram is prohibited.

## Repository Layout

```text
outputs/   AI-only guidance specifications
scripts/   installer, rollback, and smoke-test entrypoints
src/       runner, bridge, provider, credential, budget, and command modules
tests/     local smoke/unit tests
work/      iteration state generated by the review loop
```

## Current Entrypoints

```bash
scripts/install-runner.sh --dry-run
scripts/install-communication-vps.sh --dry-run --domain ai.example.com
scripts/mattermost-first-admin.sh --help
scripts/bootstrap-mattermost.sh
scripts/setup-runner-tunnel.sh --help
scripts/pair-runner.sh --help
scripts/validate-integration.sh
scripts/rollback-communication.sh --help
scripts/smoke-test.sh
python3 -m ai_remote_runner.cli status
```
